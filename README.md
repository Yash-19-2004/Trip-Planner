# Trip Planner - MERN Stack Travel Planning App

Trip Planner is a full-stack web application that helps users plan trips efficiently by creating itineraries, managing destinations, and organizing travel schedules in one place.

# Features

- Create and manage trip plans
- Add multiple destinations
- Schedule travel dates and itinerary
- Save trip data in database
- Responsive and user-friendly UI
- Secure backend APIs


# Tech Stack

# Frontend:
- React.js
- CSS, JavaScript
- Axios

### Backend:
- Node.js
- Express.js
- MongoDB
- Mongoose
